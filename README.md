## Install
Install New UI Colors from your terminal via npm. Current version is 1.2.0

```
npm i @new-ui/colors
```

To get started quickly, you can use the CDN files.
 
```html
<!-- Place this at the html head -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@new-ui/colors/dist/index.css">

<!-- Add this attribute to body -->
<body data-new-ui-theme="light">
...
```

### Available themes
- light
- light--warm
- light--cold

## Build

To build css

```
npm install
npm run build
```
