## Install:
Install New UI Colors from your terminal via npm. Current version is 0.1.0

```
npm i @new-ui/colors
```

To get started quickly, you can also use the CDN files.

Light theme: ```<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@new-ui/colors@0.1.0/dist/themes/light.css">```

Dark theme: ```<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@new-ui/colors@0.1.0/dist/themes/dark.css">```
